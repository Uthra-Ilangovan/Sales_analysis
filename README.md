📊 Project Overview

This project analyzes global sales data from a pharmaceutical manufacturing company to uncover actionable business insights across regions, distributors, customers, and sales teams.

Datamatrix-ml Pharmaceuticals operates across multiple global markets. They do not sell directly to customers but instead work through distributors who provide structured sales data in CSV format. This project transforms that raw data into an interactive analytics dashboard.

⚡ Features
Power BI Desktop for data modeling and visualization
Power Query Editor for data transformation and cleaning
Power BI Service for web-based report sharing
Multi-page interactive dashboard for deep insights
